openTPO Mark Language Instructions - 0.1.0.0_alpha

openTPO stores data as elements of a string array in JavaScript source file. Reading passage and reading questions are put together. However, these data are a little bit more than plain text.

Don't worry! The mark language is really easy for everyone to create and edit. As openTPO development team, we wish everyone join our work. Marking is not a barrier - it is in fact a good helper which would save plenty of work time!

Where are these data?

All data related to the test set live in the data folder.

Next, let's split them into parts. I'll show you how to deal with a reading passage and its questions.

Mark the reading passage

4 types of mark will be introduced. They are:

  • ||
  • ##x...##
  • >>x
  • ^^x

Here, I would like to list the sample passage again just for convenience.

Title of the sample passage

This is the first paragraph. We will mark it with an arrow.

This is the second paragraph. We will mark the phrase comic book.

This is the third paragraph. We will mark it with an arrow. And we will insert a sentence in this paragraph. Where would you put it? Do you have any idea?

Paragraph seperator: ||

Like its name, paragraph seperator simply appears when a new paragraph begins. Also, it seperates heading and first paragraph. Let's replace the newlines with "||" in our sample passage:

Title of the sample passage||This is the first paragraph. We will mark it with an arrow.||This is the second paragraph. We will mark the phrase comic book.||This is the third paragraph. We will mark it with an arrow. And we will insert a sentence in this paragraph. Where would you put it? Do you have any idea?

Highlight marker: ##

Sometimes a word or phrase or sentence in the passage should be highlighed with grey background. As we all know, highlights are associated with questions. So we should simultaneously add the number of the corresponding question.

Continue work on the sample passage. We find that Question 2 wants us to highlight the phrase "comic book". What we should do is to substitute "comic book" with "##2comic book##". If it is question 12 that needs highlight, we could similarly mark it as "##12the word##".

Attention: Question number starts from 1, and is counted only in this individual passage. Although openTPO automatically counts the questions in previous reading, marking an individual passage should start over.

Marked with "##", the sample passage would be:

Title of the sample passage||This is the first paragraph. We will mark it with an arrow.||This is the second paragraph. We will mark the phrase ##2comic book##.||This is the third paragraph. We will mark it with an arrow. And we will insert a sentence in this paragraph. Where would you put it? Do you have any idea?

Arrow indicator: >>
